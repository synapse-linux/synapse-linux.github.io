# About Synapse Linux

Synapse Linux is a community-driven Linux distribution focused on minimalism, speed, and user control.

## Philosophy

We believe in:

- **Simplicity** — clean, uncluttered design
- **Performance** — optimized kernels and minimal resource usage
- **Freedom** — full open-source commitment
- **Modernity** — rolling releases with up-to-date packages

## Team

Synapse Linux is maintained by a small, dedicated team of open-source enthusiasts.
